# Get-ScriptDirectory

Module: Capa.PowerShell.Module.Tools

## SYNOPSIS
Get the directory of the current script.

## SYNTAX

```
Get-ScriptDirectory
```

## DESCRIPTION
Get the directory of the current script.
It is better than $PSScriptRoot because it works also when running the script line by line in the console.

# Remove-CapaUnitFromReinstall

## SYNOPSIS

Remove a unit from reinstall.

## SYNTAX

### __AllParameterSets

```
Remove-CapaUnitFromReinstall [-CapaSDK] <Object> [-ComputerName] <string> [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## ALIASES

This cmdlet has the following aliases,
  {{Insert list of aliases}}

## DESCRIPTION

Remove an existing computer from reinstall in CapaInstaller.

## EXAMPLES

### EXAMPLE 1

Remove-CapaUnitFromReinstall -CapaSDK $CapaSDK -ComputerName 'PC001'

# Get-CapaProfiles
Module: Capa.PowerShell.Module.SDK.MDM

Get all profiles.

## Syntax

```powershell
Get-CapaProfiles
	-CapaSDK <Object>
```

## Description

Get all profiles from the Default Management Point.

## Examples

### Example 1
```powershell
Get-CapaProfiles -CapaSDK $CapaSDK
```
    

## Parameters

-**CapaSDK**

The CapaSDK object.
| Name | Value |
| ---- | ---- |
| Type: | Object |
| Position: | 1 | 
| Default value: | None | 
| Accept pipeline input: | false | 
| Accept wildcard characters: | false |

# Exit_ModuleNotFound
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the module was not found.

## Syntax

```powershell
Exit_ModuleNotFound
```

## Description

Uses the Exit-PpScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit_ModuleNotFound
```

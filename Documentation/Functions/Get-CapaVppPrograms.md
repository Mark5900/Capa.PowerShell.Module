# Get-CapaVppPrograms
Module: Capa.PowerShell.Module.SDK.VPP

Gets a list of all VPP programs.

## Syntax

```powershell
Get-CapaVppPrograms
	-CapaSDK <Object>
```

## Description

Gets a list of all VPP programs.

## Examples

### Example 1
```powershell
Get-CapaVppPrograms -CapaSDK $CapaSDK
```
    

## Parameters

-**CapaSDK**

The CapaSDK object.
| Name | Value |
| ---- | ---- |
| Type: | Object |
| Position: | 1 | 
| Default value: | None | 
| Accept pipeline input: | false | 
| Accept wildcard characters: | false |

# Add-CapaPackageToBusinessUnit

## SYNOPSIS

Adds a package to a business unit.

## SYNTAX

### __AllParameterSets

```
Add-CapaPackageToBusinessUnit [-CapaSDK] <Object> [-PackageName] <string> [-PackageVersion] <string>
 [-PackageType] <string> [-BusinessUnitName] <string> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ALIASES

This cmdlet has the following aliases,
  {{Insert list of aliases}}

## DESCRIPTION

Adds a package to a business unit.

## EXAMPLES

### EXAMPLE 1

Add-CapaPackageToBusinessUnit -CapaSDK $CapaSDK -PackageName 'Winrar' -PackageVersion 'v3.0' -PackageType Computer -BusinessUnitName 'HeadQuarterBronx'

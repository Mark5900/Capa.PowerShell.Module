# Get-CapaUsers
Module: Capa.PowerShell.Module.SDK.User

Get a list of all users.

## Syntax

```powershell
Get-CapaUsers
	-CapaSDK <Object>
```

## Description

Get a list of all users.

## Examples

### Example 1
```powershell
Get-CapaUsers -CapaSDK $CapaSDK
```
    

## Parameters

-**CapaSDK**

The CapaSDK object.
| Name | Value |
| ---- | ---- |
| Type: | Object |
| Position: | 1 | 
| Default value: | None | 
| Accept pipeline input: | false | 
| Accept wildcard characters: | false |

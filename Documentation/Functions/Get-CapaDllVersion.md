# Get-CapaDllVersion
Module: Capa.PowerShell.Module.SDK.Container

Get the version of the CapaSDK dll.

## Syntax

```powershell
Get-CapaDllVersion
	-CapaSDK <Object>
```

## Description



## Examples

### Example 1

```powershell
Get-CapaDllVersion
```

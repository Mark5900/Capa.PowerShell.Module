# Exit_CommandNotDelivered
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the command was not delivered.

## Syntax

```powershell
Exit_CommandNotDelivered
```

## Description

Uses the Exit-PSScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit_CommandNotDelivered
```

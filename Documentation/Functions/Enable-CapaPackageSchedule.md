# Enable-CapaPackageSchedule
Module: Capa.PowerShell.Module.SDK.Package

Enable a packages schedule.

## Syntax

```powershell
Enable-CapaPackageSchedule
	-CapaSDK <Object>
	-PackageName <String>
	-PackageVersion <String>
	-PackageType <String>
```

## Description

Eanble a packages schedule if a schedule is set.

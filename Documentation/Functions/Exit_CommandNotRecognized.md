# Exit_CommandNotRecognized
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the command is not recognized.

## Syntax

```powershell
Exit_CommandNotRecognized
```

## Description

Uses the Exit-PpScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit_CommandNotRecognized
```

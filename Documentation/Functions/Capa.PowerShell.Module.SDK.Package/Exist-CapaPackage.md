# Exist-CapaPackage

## SYNOPSIS

Verifies if a package exists.

## SYNTAX

### __AllParameterSets

```
Exist-CapaPackage [-CapaSDK] <Object> [-Name] <string> [-Version] <string> [-Type] <string>
 [<CommonParameters>]
```

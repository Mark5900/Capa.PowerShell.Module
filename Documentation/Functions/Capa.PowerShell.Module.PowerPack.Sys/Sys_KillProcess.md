# Sys_KillProcess

## SYNOPSIS

Kills a process.

## SYNTAX

### __AllParameterSets

```
Sys_KillProcess [-ProcessName] <string> [<CommonParameters>]
```

## DESCRIPTION

This function kills a process by its name.

### EXAMPLE 1

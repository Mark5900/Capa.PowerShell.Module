# Exit_CommandHandlingFailed
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the command handling failed.

## Syntax

```powershell
Exit_CommandHandlingFailed
```

## Description

Uses the Exit-PSScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit_CommandHandlingFailed
```

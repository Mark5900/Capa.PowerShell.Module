# Exit-PpPackageCancelled
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the package is cancelled.

## Syntax

```powershell
Exit-PpPackageCancelled
```

## Description

Uses the Exit-PpScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit-PpPackageCancelled
```

# Exit-PpMissingDiskSpace
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that there is missing disk space.

## Syntax

```powershell
Exit-PpMissingDiskSpace
```

## Description

Uses the Exit-PpScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit-PpMissingDiskSpace
```

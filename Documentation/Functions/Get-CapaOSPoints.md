# Get-CapaOSPoints
Module: Capa.PowerShell.Module.SDK.OSDeployment

Gets a list of OS Points.

## Syntax

```powershell
Get-CapaOSPoints
	-CapaSDK <Object>
```

## Description

Gets a list of OS Points.

## Examples

### Example 1
```powershell
Get-CapaOSPoints -CapaSDK $CapaSDK
```
    

## Parameters

-**CapaSDK**

The CapaSDK object.
| Name | Value |
| ---- | ---- |
| Type: | Object |
| Position: | 1 | 
| Default value: | None | 
| Accept pipeline input: | false | 
| Accept wildcard characters: | false |

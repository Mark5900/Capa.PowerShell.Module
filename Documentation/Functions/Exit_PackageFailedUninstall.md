# Exit_PackageFailedUninstall
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the package failed to uninstall.

## Syntax

```powershell
Exit_PackageFailedUninstall
```

## Description

Uses the Exit-PpScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit_PackageFailedUninstall
```

## Syntax

```powershell
Add-CapaUnitToPackage
	-CapaSDK <Object>
	-PackageType <String>
	-PackageName <String>
	-PackageVersion <String>
	-UnitName <String>
	-UnitType <String>
```

A detailed description of the Add-CapaUnitToPackage function.

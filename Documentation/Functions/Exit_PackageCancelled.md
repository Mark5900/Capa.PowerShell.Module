# Exit_PackageCancelled
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the package is cancelled.

## Syntax

```powershell
Exit_PackageCancelled
```

## Description

Uses the Exit-PSScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit_PackageCancelled
```

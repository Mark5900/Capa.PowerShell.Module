# File_DelTree
Module: Capa.PowerShell.Module.PowerPack.File

Delete a folder.

## Syntax

```powershell
File_DelTree
	-Path <String>
```

## Description



## Examples

### Example 1
```powershell
File_DelTree -Path "C:\Temp\test"
```
    

## Parameters

-**Path**

The folder to delete.
| Name | Value |
| ---- | ---- |
| Type: | String |
| Position: | 1 | 
| Default value: | None | 
| Accept pipeline input: | false | 
| Accept wildcard characters: | false |

# Exit_CommandSucceded
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the command succeded.

## Syntax

```powershell
Exit_CommandSucceded
```

## Description

Uses the Exit-PSScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit_CommandSucceded
```

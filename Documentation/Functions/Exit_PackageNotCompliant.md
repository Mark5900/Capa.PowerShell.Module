# Exit-PpPackageNotCompliant
Module: Capa.PowerShell.Module.PowerPack.Exit

Set error code that the package is not compliant.

## Syntax

```powershell
Exit-PpPackageNotCompliant
```

## Description

Uses the Exit-PpScript that comes from PSlib.psm1, to set the package error.

## Examples

### Example 1
```powershell
Exit-PpPackageNotCompliant
```

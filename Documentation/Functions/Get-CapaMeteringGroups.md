# Get-CapaMeteringGroups
Module: Capa.PowerShell.Module.SDK.Inventory

Get metering groups.

## Syntax

```powershell
Get-CapaMeteringGroups
	-CapaSDK <Object>
```

## Description

Gets a list of metering groups.

## Examples

### Example 1
```powershell
Get-CapaMeteringGroups -CapaSDK $CapaSDK
```
    

## Parameters

-**CapaSDK**

The CapaSDK object.
| Name | Value |
| ---- | ---- |
| Type: | Object |
| Position: | 1 | 
| Default value: | None | 
| Accept pipeline input: | false | 
| Accept wildcard characters: | false |

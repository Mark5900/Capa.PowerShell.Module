# Get-CapaBusinessUnits
Module: Capa.PowerShell.Module.SDK.SystemSdk

Get a list of all business units.

## Syntax

```powershell
Get-CapaBusinessUnits
	-CapaSDK <Object>
```

## Description

Get a list of all business units.

## Examples

### Example 1
```powershell
Get-CapaBusinessUnits -CapaSDK $CapaSDK
```
    

## Parameters

-**CapaSDK**

The CapaSDK object.
| Name | Value |
| ---- | ---- |
| Type: | Object |
| Position: | 1 | 
| Default value: | None | 
| Accept pipeline input: | false | 
| Accept wildcard characters: | false |

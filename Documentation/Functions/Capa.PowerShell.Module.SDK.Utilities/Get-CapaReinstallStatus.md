# Get-CapaReinstallStatus

## SYNOPSIS

Gets the reinstall status for a unit.

## SYNTAX

### __AllParameterSets

```
Get-CapaReinstallStatus [-CapaSDK] <Object> [-UnitName] <Object> [-UnitType] <Object>
 [<CommonParameters>]
```

# Remove-CapaUnitByUUID

## SYNOPSIS

Remove a unit by UUID.

## SYNTAX

### __AllParameterSets

```
Remove-CapaUnitByUUID [-CapaSDK] <Object> [-UUID] <string> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## ALIASES

This cmdlet has the following aliases,
  {{Insert list of aliases}}

## DESCRIPTION

Delete an existing unit by UUID in CapaInstaller.

## EXAMPLES

### EXAMPLE 1

Remove-CapaUnitByUUID -CapaSDK $CapaSDK -UUID '12345678-1234-1234-1234-123456789012'
